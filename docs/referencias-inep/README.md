# Documentação Oficial de Referência - INEP / ENEM

Esta pasta contém os manuais de correção oficiais do INEP que servem de base para o Corretor de Redação ENEM AI.

## Estrutura sugerida:
- `competencia-1.pdf`: Norma Culta e Ortografia.
- `competencia-2.pdf`: Compreensão do tema e Repertório Sociocultural.
- `competencia-3.pdf`: Organização e Intencionalidade do texto.
- `competencia-4.pdf`: Coesão e Conectivos.
- `competencia-5.pdf`: Proposta de Intervenção.

## Como utilizar:
Ao adicionar um manual aqui, peça para a IA ler e atualizar o "MASTER_ENEM_PROMPT" no arquivo `src/lib/constants.ts` com as novas tabelas de pontuação refinadas.
