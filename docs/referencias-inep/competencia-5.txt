# COMPETÊNCIA 5: Proposta de Intervenção

## ELEMENTOS OBRIGATÓRIOS (40 PONTOS CADA):
1. **Ação:** O que será feito?
2. **Agente:** Quem fará? (Governo, Família, Escola...)
3. **Meio/Modo:** Como será feito?
4. **Efeito/Finalidade:** Para que será feito?
5. **Detalhamento:** Explicação extra sobre um dos outros elementos.

## GRADE DE PONTUAÇÃO:
- **200 PONTOS:** Contemplar os 5 elementos (Ação + Agente + Meio + Efeito + Detalhamento).
- **160 PONTOS:** Contemplar 4 elementos.
- **120 PONTOS:** Contemplar 3 elementos.
- **80 PONTOS:** Contemplar 2 elementos.
- **40 PONTOS:** Contemplar 1 elemento.
- **0 PONTOS:** Não apresentar proposta ou desrespeitar os direitos humanos.
