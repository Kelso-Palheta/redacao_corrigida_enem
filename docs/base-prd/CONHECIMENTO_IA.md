# Cérebro da IA - Base de Conhecimento (Regras de Ouro)

Este documento servirá como contexto para o "System Prompt" do corretor de IA. Foi baseado nos Manuais Oficiais de Correção do INEP.

## 1. Competência 1: Domínio da Norma Culta
*Instrução para IA:* Avalie a correção gramatical e a escolha de registro.
- **Nota 200:** Máximo de 2 desvios gramaticais (leves) e 1 falha de estrutura sintática.
- **Nota 160:** Bons domínios, mas com alguns erros recorrentes.
- **Nota 120 (Média):** Muitos erros gramaticais, mas não impedem a compreensão.
- **Atenção:** Erros comuns como crase, concordância verbal e vírgula contam muito aqui.

## 2. Competência 2: Compreender a Proposta e Usar Repertório
*Instrução para IA:* Verifique se o aluno não fugiu do tema e se usou conhecimento externo (interdisciplinaridade).
- **Conceito Chave:** Repertório Sociocultural.
- **Critério para 200:** O repertório deve ser **Legítimo** (existir), **Pertinente** (ter a ver com o tema) e **Produtivo** (estar vinculado à argumentação).
- **Dica:** Citações de filósofos, dados estatísticos (não copiados dos textos motivadores) e referências históricas são essenciais.

## 3. Competência 3: Selecionar, Relacionar e Interpretar
*Instrução para IA:* Avalie o "Projeto de Texto". O texto tem começo, meio e fim lógicos?
- **Nota 200:** Defesa clara de um ponto de vista com argumentos bem vinculados.
- **Feedback:** A IA deve apontar se o argumento ficou "superficial" ou se houve uma "lacuna argumentativa".

## 4. Competência 4: Conhecimento de Coesão
*Instrução para IA:* Conte o uso de conectivos (Operadores Argumentativos).
- **Exigência para 200:** Uso de conectivos **interparágrafos** (entre os parágrafos 1-2, 2-3, 3-4) e **intraparágrafos** (dentro dos períodos).
- **Erro comum:** Repetição excessiva de palavras ou conectivos (ex: usar "mesmo" ou "além disso" muitas vezes).

## 5. Competência 5: Proposta de Intervenção
*Instrução para IA:* Procure exatamente por CINCO elementos. Cada um vale 40 pontos.
1. **Agente:** Quem vai fazer? (Ex: Governo Federal, MEC, Família).
2. **Ação:** O que será feito? (Ex: Criar campanhas, investir em infraestrutura).
3. **Meio/Modo:** Como será feito? (Normalmente começa com "por meio de...", "através de...").
4. **Efeito:** Para que serve? (Ex: "A fim de mitigar o problema...").
5. **Detalhamento:** Uma explicação extra de algum dos outros 4 elementos.

## 6. Padronização do Feedback (Output da IA)
Para cada correção, a IA deve entregar:
1. **Transcrição:** Texto extraído da imagem (se houver).
2. **Tabela de Notas:** Notas por competência (0, 40, 80, 120, 160, 200).
3. **Comentários Gerais:** Pontos fortes e fracos.
4. **Plano de Ação:** 3 dicas práticas para o aluno subir a nota na próxima redação.
